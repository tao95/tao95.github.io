一.搭建的大致步骤

（详细的操作步骤，网上多如牛毛，我在这里就不细说了）

1.建立github账号，创建xxx.github.io的仓库。
2.安装github的客户端。
3.翻墙、翻墙、翻墙。（因为在安装hexo的时候，不翻墙的话，很难安装成功）
4.安装相应的npm、node.js和hexo等(不会的，找度娘)。
5.在自己的电脑上（windows）新建文件夹,命名为blog，然后管理员运行命令行，进入blog文件夹。
6.运行命令: npm install -g hexo
7.接着初始化hexo ,hexo init
8.然后用命令hexo g生成静态页面
9.最后使用命令hexo s启动服务,在浏览器中输入localhost:4000 就可以看到效果
注：a.运行cmd是要用管理员身份运行（win+x+a）。
b.在安装hexo时,连接一下vpn,可能要快点，不然会一直卡在那儿。
c.hexo常用命令：
hexo n “hello world” #新建一篇文章
hexo n page “photo” #新建一个相册页面
hexo g #生成静态页面到public目录（这个目录就是上传到 github的文件）
hexo s #开启预览访问服务（默认端口4000，’ctrl + c’关闭 server）
hexo d #部署到GitHub
hexo clean #用来清除public文件夹
